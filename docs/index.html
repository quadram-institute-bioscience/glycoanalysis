Getting Started

GlycoCrab is a suite of tools designed to streamline the analysis of glycan mass spectrometry data. The workflow typically starts with GlycoPrep to process raw data into a structured format.

Using GlycoPrep
  • Input: Requires 3 Excel files:
    1. Raw Peaks (one sheet per sample)
    2. Metadata (must have 'sample_sheet' column)
    3. Glycan Database (Mass, Composition)
  • Process: Click "Process Files" to match peaks and calibrate.
  • Output: Download matched_glycans.tsv for downstream analysis.
